# Ejercicios de Estructuras de Datos: Pilas (Stacks)

Este proyecto contiene la resolución de dos ejercicios clásicos utilizando la estructura de datos lineal **Pila (Stack)** en el lenguaje C#.

## Contenido del Repositorio
1. **Verificación de Paréntesis:** Un algoritmo que utiliza una pila para determinar si una expresión matemática tiene sus signos de agrupación `{ [ ( ) ] }` correctamente balanceados.
2. **Torres de Hanoi:** Resolución del rompecabezas lógico mediante un algoritmo recursivo que simula el movimiento de discos entre tres torres (Origen, Auxiliar, Destino).
